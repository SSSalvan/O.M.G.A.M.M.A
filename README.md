# O.M.G.A.M.M.A Game Development

## Overview
An interactive game development project built from scratch. It features custom game mechanics, state management, and physics implementation. This project was developed as part of a collaborative game jam/coursework to explore the fundamentals of game engine logic.

## Technical Details
- **Environment:** Java / OOP Paradigms
- **Graphics:** Custom Rendering Loop
- **Physics:** 2D Collision and Kinematics

## Features
- Robust game loop architecture.
- Custom asset rendering and event listeners.
- Score tracking and dynamic difficulty scaling.
